dotfiles
========

dotfiles managed using [homesick](https://github.com/technicalpickles/homesick).

### Installation

1. Setup homesick and symlick dotfiles:
```
    gem install homesick
    homesick clone mattmenefee/dotfiles
    homesick symlink dotfiles
```
2. Install Vundle
```
    cd ~/.dotfiles
    zsh init.zsh
```
